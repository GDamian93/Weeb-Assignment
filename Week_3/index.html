Using CSS

  • In-line CSS: Styles applied directly inside an HTML tag using the style attribute.

  • Internal CSS: Styles defined inside the HTML page, within the style tag in the head section.

  • External CSS: Styles stored in a separate file.css, linked to the HTML page using the link tag. This is the most organized way.

Selectors

  • Element Selector: Targets HTML elements directly (e.g., p, h1, div).

  • Class Selector: Targets elements with a specific class attribute (e.g., .classname).

  • ID Selector: Targets a unique element with a specific id attribute (e.g., #idname).

  • Universal Selector: Targets all elements (e.g., *).

Box Model

  • Content: The actual content of the box, where text and images appear.

  • Padding: The space between the content and the border, inside the box.

  • Border: A border that surrounds the padding (if any) and content.

  • Margin: The space outside the border, separating the box from other elements.

Conclusion

Understanding CSS is crucial for web development. It allows you to create visually appealing and well-structured web pages.
